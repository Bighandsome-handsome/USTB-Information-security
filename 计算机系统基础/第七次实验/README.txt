计算机系统基础第七次实验
作业满分：6.00，共 3道题
破解代码使用说明：
（1）静态链接练习-01和-02：学号正常按照题目要求输入你的学号，string ans输入"f"即可通过测试；
（2）静态链接练习-03使用说明：
         1. 打开文件夹中的step 3.cpp文件
         2. 按照题目要求输入你的学号后四位，注意是字符串输入，如后四位1111应输入
                                           char studentid[] ="1111"；
         3. passwd[ ] 随便输入一个字符串，建议复制studentid[]的内容
         4. string ans输入"f"
         5. 用编译器运行程序（不建议使用Dev C编译，会报错）
         6. 运行后你会得到一个字符串和"Error Ans0"
         7. 将运行后得到的字符串（应该是一个四位整数）复制到passwd[ ]即可
         8. 再次运行程序，检验是否正确。若正确会看到"Accurate Ans"的输出
